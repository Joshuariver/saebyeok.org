# 노래모임 새벽 디지털 아카이브 (Saebyeok Digital Archive)

이곳은 한국 현대 음악사의 소중한 자산인 **'노래모임 새벽'**의 활동 기록과 음원을 보존하고 공유하기 위한 디지털 공간입니다. 

본 프로젝트는 노래모임 새벽의 정신을 기리고, 그들의 음악을 사랑하는 모든 이들이 당시의 울림을 온전하게 다시 경험할 수 있도록 돕기 위해 구축되었습니다.

---

## 🎧 아카이브 소개
본 사이트는 '노래모임 새벽'의 음원을 중심으로 한 **단순 감상용 아카이브**입니다. 
당시의 앨범 아트워크, 가사, 그리고 시대의 정서가 담긴 음원들을 연도별/앨범별로 정리하여 제공합니다.

- **주요 기능**: 음원 스트리밍 서비스, 앨범별 가사 및 정보 확인, 활동 연혁 조회
- **기술 스택**: HTML5, CSS3, JavaScript (Vanilla JS), GitHub Pages
- **데이터 보관**: 음원 및 대용량 미디어 파일은 외부 클라우드 저장소를 통해 안전하게 관리됩니다.
